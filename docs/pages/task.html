Write a program to reverse a string in JavaScript

The task is to write a JavaScript program that inverts a string. For example, if the string "Hello" is given, the program should return the string "olleH"...

The string reversal algorithm can be implemented as follows:

  • Create a function that takes an input string as an argument.
  • Initialise the variable reversedString (or any other variable name) that will store the reversed string. Set the value of this variable as an empty string.
  • Using a loop, iterate through the original string in reverse order, starting with the last character.
  • At each iteration, add the current character to the beginning of the reversedString.
  • When the loop ends, reversedString will contain the reversed string.
  • Return reversedString from the function.
